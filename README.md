# Анализ успеваемости студентов

## Цель проекта
Проект исследует, как учебные привычки, цифровое поведение и психическое здоровье студентов влияют на их продуктивность и результаты экзаменов. Используются SQL-запросы для анализа, группировок и выявления закономерностей.

## Источник данных
Набор синтетических данных о студентах (5000 записей), включающий: возраст, пол, академический уровень, учебные часы, цифровые привычки, здоровье и результаты экзаменов.

## Структура таблиц
- **students** — информация о студентах: `student_id`, `age`, `gender`, `academic_level`
- **habits** — учебные привычки и поведение: `study_hours`, `self_study_hours`, `online_classes_hours`, `social_media_hours`, `gaming_hours`, `sleep_hours`, `screen_time_hours`, `exercise_minutes`, `caffeine_intake_mg`, `part_time_job`, `upcoming_deadline`, `internet_quality`
- **health** — психическое и физическое здоровье: `mental_health_score`, `focus_index`, `burnout_level`
- **performance** — продуктивность и успеваемость: `productivity_score`, `exam_score`

## Примеры запросов

### 1. Общее количество студентов
```sql
SELECT COUNT(*) AS total_students
FROM students;
```
Результат: 5000

### 2. Средний балл по экзаменам
```sql
SELECT AVG(exam_score) AS avg_exam_score
FROM performance;
```
Результат: 78.4

### 3. Топ-10 студентов по экзаменам
```sql
SELECT s.student_id, s.age, s.gender, p.exam_score
FROM students s
JOIN performance p ON s.student_id = p.student_id
ORDER BY p.exam_score DESC
LIMIT 10;
```
Результат: показывает 10 студентов с наивысшими баллами по экзаменам

### 4. Связь учебных часов и продуктивности
```sql
SELECT h.study_hours, AVG(p.productivity_score) AS avg_productivity
FROM habits h
JOIN performance p ON h.student_id = p.student_id
GROUP BY h.study_hours
ORDER BY h.study_hours;
```

### 5. Влияние сна на продуктивность
```sql
SELECT h.sleep_hours, AVG(p.productivity_score) AS avg_productivity
FROM habits h
JOIN performance p ON h.student_id = p.student_id
GROUP BY h.sleep_hours
ORDER BY h.sleep_hours;
```

### 6. Влияние социальных сетей на продуктивность
```sql
SELECT h.social_media_hours, AVG(p.productivity_score) AS avg_productivity
FROM habits h
JOIN performance p ON h.student_id = p.student_id
GROUP BY h.social_media_hours
ORDER BY h.social_media_hours;
```

# Вывод проделанной работы

В ходе проекта была проведена комплексная работа по анализу данных студентов, включая подготовку, структурирование и исследование взаимосвязей между учебными привычками, здоровьем и успеваемостью.

## 1. Разделение исходной таблицы

Исходная таблица `full_data` была разделена на четыре логические таблицы:

- **students** — базовая информация о студентах;  
- **habits** — учебные привычки и поведение;  
- **health** — показатели психического и физического здоровья;  
- **performance** — результаты экзаменов и продуктивность.  

**Эффект:** данные стали структурированными, удобно доступными для анализа, с логическими связями через `student_id`.

## 2. Анализ базовой статистики

- **Количество студентов:** 5000  
- **Средний возраст:** 20 лет  
- **Средние учебные часы:**  
  - `study_hours`: 4.54  
  - `self_study_hours`: 2.48  
  - `online_classes_hours`: 2.01  
- **Средний уровень выгорания и концентрации:**  
  - `burnout_level`: 45.62  
  - `focus_index`: 29.43  
- **Средний балл по экзаменам и продуктивности:**  
  - `exam_score`: 18.80  
  - `productivity_score`: 37.27  

## 3. Взаимосвязь привычек, здоровья и продуктивности

- **Учебные часы и продуктивность:**  
  Даже небольшое вовлечение повышает результат, максимум ~30–34 при 1.5–2 часах.  
- **Выгорание и экзамены:**  
  Линейной зависимости нет; влияние многогранное и зависит от концентрации и учебных привычек.  
- **Топ-10 студентов:**  
  Демонстрирует разнообразие пола и возраста; высокие баллы чаще совпадают с высокой продуктивностью.  
- **Академический уровень:**  
  Мало влияет на средние результаты и учебную активность.

## 4. Влияние сна на продуктивность

- Оптимальная продуктивность достигается при 4–5 часах сна, с пиками до 58 баллов.  
- Очень малый сон (<4.1 часа) снижает продуктивность; на результаты влияют и другие факторы.

## 5. Влияние социальных сетей

- Умеренное время в соцсетях (0–0.2 часов) иногда связано с высокой продуктивностью, но встречаются резкие выбросы.  
- Средняя продуктивность при типичном использовании соцсетей — 30–45 баллов.  
- Четкой зависимости нет, влияние комбинируется с другими привычками (сон, учебные часы).
